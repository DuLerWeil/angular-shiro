authenticator
service in module angularShiro.services

Description

Service in charge of the authentication process.

Default implementation send a post request to the uri /api/authenticate with a UsernamePasswordToken as data.

This service is not intended to be accessed directly; It's meant to be used internally by the subject service.

Configuration

In case that the default behavior of the authenticator service does not fit your needs you can - specify our own uri (if that the point of contention)

app.config([ 'authenticatorProvider', function(authenticatorProvider) {
	authenticatorProvider.setUri('path/to/my/authentication/service');
} ]);
